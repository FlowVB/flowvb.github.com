FlowVB by FlowVB

Gating Flow Cytometry Data with Variational Bayes

About

FlowVB is an algorithm for the clustering of flow cytometry data. It uses a mixture of Student-t distributions which are fit using variational Bayes. Our algorithm is based on Archambeau & Verleysen 2006, "Robust Bayesian Clustering". FlowVB was presented at the FlowCAP I Summit 2010 on September 21, 2010. Please read the abstract and presentation.

For FlowCAP I we used a prototype implementation in MATLAB. You may download the MATLAB implementation, but please be advised that this is untested software and will not be further developed.

We are reimplementing the algorithm in Python and are adding more features. Currently this project is in early alpha mode, but already functional. Abobe you may download the source. Minimum requirements to run FlowVB are:

  • Python 2.6.5
  • Numpy 1.4.0
  • Scipy 0.8.0b1
  • Matplotlib 0.99.3
  • Enthought Traits API 3.4.0
  • Cython 0.12.1
  • A C-compiler
